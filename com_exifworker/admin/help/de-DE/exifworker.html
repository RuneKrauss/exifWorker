Liste

Hier haben Sie einen Überblick über Ihre Bilder, die sie hochgeladen haben.

Löschen

Wählen Sie die Bilder aus, die Sie löschen möchten. Zu Ihrer Sicherheit müssen Sie den Vorgang jedoch noch einmal bestätigen. Mit der Bestätigung wird sowohl das Bild als auch der Eintrag gelöscht.

Bearbeiten

Hier gelangen Sie in den Bereich, wo Sie die Exif-Daten der Bilder einsehen können. Exchangeable Image File Format (Exif) ist ein Format, welches Metadaten in JPG-Dateien speichert. Dazu zählen GPS-Daten, Kameramodell, Hersteller usw. Sie können diese Daten bearbeiten oder komplett bereinigen lassen.

Neu

Unter dieser Rubrik können Sie ein neues Bild hinzufügen und einen Namen vergeben. Es ist dabei nicht zulässig, mehrere Bilder unter dem gleichen Dateinamen hochzuladen. In einem solchen Fall passen Sie bitte die jeweiligen Dateinamen an.

Optionen

Sie können diese Komponente individuell anpassen. So können Sie z. B. einstellen, ob alle EXIF-Daten im Bearbeitungsbereich angezeigt werden sollen oder nicht. Zudem können Sie die Berechtigungen anpassen. Es ist also möglich, z. B. einem Admin den Zugriff auf bestimmte Schaltflächen zu verbieten.
