List

Here, you have an overview on the images you have uploaded.

Delete

Choose images which you want to delete. For your safety, you have to confirm the action. After your confirmation, the image and entry will be deleted.

Edit

In this area, you are able to see the exif data of the images. Exchangeable Image File Format (Exif) is a standard that specifies the formats for images, sound, and ancillary tags used by digital cameras (including smartphones), scanners and other systems handling image and sound files recorded by digital cameras. You can edit and clear up these data.

New

In this section, you are able to add an image with a name. It is not allowed to upload several images with the same name. In that case, please adapt the name of these files.

Options

You can adjust this component individually. For example, you are able to adjust whether all exif data are displayed in the editing area or not. Moreover, the permissions can be adjusted. It is also possible to prohibit specific access for an admin regarding buttons with an action.
